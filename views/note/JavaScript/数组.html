数组总结

Hans 2021-08-18 JavaScript

# 概念

  • 数组也属于对象,是一个有序的集合
  • 数组中可以存储任意数据类型
  • 可以是栈或者队列来进行数组的操作
  • arr.length实际上是数组最大索引值+1
  • 通过构造函数创建数组 new Array(num),会存储num个值为undefined的数组
  • 数组在进行原始类型转化时会调用toString()进行转化,对于空数组[]会转化为空串""
  • 数组之间的比较由于会转化为原始类型例如:0 == [],所以数组之间的比较不使用==
  • 数组通过操作length属性能对数组进行截取，访问超出部分返回undefind,清空数组的方式length=0

Tips: 请将数组视为作用于 有序数据 的特殊结构。它们为此提供了特殊的方法。数组在 JavaScript 引擎内部是经过特殊调整的，使得更好地作用于连续的有序数据，所以请以正确的方式使用数组。如果你需要任意键值，那很有可能实际上你需要的是常规对象{}。

# 遍历数组

遍历数组的方法可以使用常规的for循环遍历,若只需要遍历数组的value也可以使用for..of进行遍历,当然数组中也可以像对象一样使用for..in来遍历数组的key,数组的key就是index,但是for..in 循环会遍历所有属性，不仅仅是这些数字属性,并且for..in对遍历对象有做特殊优化,在数组中会慢10~100倍,因此不推荐使用for..in遍历数组

# forEach

forEach遍历允许为每个项目执行一个函数

const arr =[1,2,3];
arr.forEach((item,index,arr) =>{
  console.log(item);  //1,2,3
  console.log(index);  //0,1,2
  console.log(arr);//[1,2,3]  
})

该函数的结果（如果它有返回）会被抛弃和忽略

const arr =[1,2,3];
let result = arr.forEach((item,index,arr) =>{
  return 1;
})
console.log(result);//undefined

# 数组方法

JS中提供了很多操作数组的方法,在这里我们将常用方法进行分类

  • 原数组上操作的为一类
  • 返回新数组的为一类

# 在原数组上操作

# push/pop

在原数组尾部添加/删除元素,

  • push可以在尾部添加多个元素（包含任意数据类型）
  • pop不支持删除多个，仅删除最后一个元素
const arr = [0,1,2,3,4];
arr.push(5,6);//[0,1,2,3,4,5,6]
arr.pop();//[1,2,3,4,5,6]

# shift/unshit

在原数组头部添加/删除数组元素

  • shift删除头部第一个元素，不支持删除多个
  • unshift可以在头部添加多个元素（包含任意数据类型）
const arr = [2,3,4,5];
arr.shift();//[3,4,5]
arr.unshift(0,1,2);//[0,1,2,3,4,5]

pop和push在执行效率上会比unshift和shift更高,它们在操作数组时,只需关注数组最后一项,并在后面继续添加或删除元素,在更新length属性,而unshift和shift需要对头部元素进行操作,后面的元素索引也会重排,最后在更新length属性,效率上会更低

# splice

在删除数组元素时也可以像对象一样使用delete进行删除数组中的元素，但这种方式删除不会使数组进行重排，被删除的值变成了undefind,对此使用数组方法splice进行操作数组，并且数组会重新排列，并更新length

const arr = [1,2,3];
delete arr[0];//[undefind,2,3]

使用splice删除元素，splice(index,deleteCount),index为当前操作元素索引，deleteCount为当前元素后删除几位元素（1的话为删除自身）,执行完毕会返回被删除的元素,只传入index默认删到底

const arr = [1,2,3,4];
arr.splice(0,1);//删除自身
console.log(arr);//[2,3,4]
arr.splice(1,2);//[2]

当前位置删除元素后在添加新元素splice(index,deleteCount,newEle),newEle可添加多个，支持任意类型

const arr = [1,2,3];
arr.splice(1,1,4,5);//[1,4,5,3]
arr.splice(1,2,7,8);//[1,7,8,3]

同样也可以不删除元素在指定位置前面添加元素

const arr = [1,2,3];
arr.splice(1,0,4,5);//[1,4,5,2]

支持index的值为负数，则由右往左进行操作(最右边从-1开始计算元素位置)

const arr = [1,2,3,4,5];
arr.splice(-1,2,7,8);//[1,2,4,7,8]
arr.splice(-3,0,9);//[1,2,9,4,7,8]

# sort(fn)

对数组进行 原位（in-place） 排序，更改元素的顺序,同时会返回排序后的新数组,默认是按字符串进行排序,可传入排序规则
排序规则

function compare(a, b) {
  if (a > b) return 1; // 如果第一个值比第二个值大
  if (a == b) return 0; // 如果两个值相等
  if (a < b) return -1; // 如果第一个值比第二个值小
}```

### 返回新数组  

#### slice  

`slice`方法会对数组进行切割，`slice(start,end)`,从`start`开始包括`start`到`end`结束，不包括`end`,若只传`start`则默认从`strat`位置裁切到结尾，支持负向索引，并返回裁剪好的新数组  

```js
let arr = ["t", "e", "s", "t"];
alert( arr.slice(1, 3) ); // e,s（复制从位置 1 到位置 3 的元素）
alert( arr.slice(-2) ); // s,t（复制从位置 -2 到尾端的元素）

# concat

创建一个新数组，其中包含来自于其他数组和其他项的值(接受元素或者数组)。

const arr = [1,2];
const newArr = [5,6]
arr.concat(3,4);//[1,2,3,4]
arr.concat(newArr);//[1,2,3,4,5,6]

**Tips:**类数组也会被当成对象被复制到新数组,可通过设置Symbol.isConcatSpreadable属性,那它就会被当成数组来进行复制

const arr = [1,2];
const arrayLike={
  0:'a',
  1:'b',
  [Symbol.isConcatSpreadable]:true,
  length:2
}
arr.concat(arrayLike);//[1,2,'a','b'];

# map

它对每个元素调用方法,并返回结果数组

let result = arr.map(function(item, index, array) {
  // 返回新值而不是当前元素
})

let arr = [1,2,3];
let result = arr.map((item)=>item*2);
console.log(arr);//[1,2,3]
console.log(result);//[1,4,6]

# 数组搜索

# indexOf/lastIndexOf和includes

  • indexdOf(item,from) 从左到右检索item,若存在返回第一个匹配值的索引,若无匹配结果返回-1,form可以指定开始查找的索引位置
  • lastIndexdOf(item,from) 从右到左检索item,若存在返回第一个匹配值的索引,若无匹配结果返回-1,from可以指定开始查找的索引位置
  • includes(item,from))从右到左检索item,若存在返回true,若无匹配结果返回false,from可以指定开始查找的索引位置
  • 同时from接受索引为负数 这些方法使用严格相等进行匹配,includes和indexOf的区别还存在于includes可以辨别NaN

# find 和 findIndex

find接受更复杂的查找方式,可以传入一个函数进行匹配,返回true的第一个（单个）元素

const arr = [{name:'Lee'},{name:'John'},{name:'Lee'}];
arr.find((item,index,arr)=>{
  return item.name = 'Lee';
})

findIndex则是反回匹配元素的索引,未找到匹配项都是返回-1

# filter

若需要匹配多项,则使用使用 arr.filter(fn)。,会将符合条件的项生成一个新数组,若无匹配项返回空数组

const arr = [{name:'Lee'},{name:'John'},{name:'Lee'}];
let newArr = arr.filter((item,index,arr) => item.name ==='Lee');
console.log(newArr)
